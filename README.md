# 基本介绍
#### 此项目主要使用J2EE相关知识进行搭建。<br>
#### 前端框架使用Bootstrap,前端页面使用JQUERY和jsp开发。<br>
#### 后端使用了MVC分层设计模式开发，有校验是否登录功能。<br>
#### 数据都是从数据库中获取显示。
